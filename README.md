# Office Automation SPA

## Overview
This project is a demo implementation of an Office Correspondence System built with Angular.

## Features
- Login with mock authentication
- AuthGuard protected routes
- Inbox / Outbox / Notes
- Compose with Reply / Forward
- File attachment simulation
- LocalStorage token handling

## Tech Stack
- Angular
- TypeScript
- RxJS
- SCSS

## How to Run
npm install
ng serve
